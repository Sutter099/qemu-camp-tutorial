# 项目阶段

恭喜各位同学顺利通过专业阶段，相信大家已经具备了开发 QEMU 的能力，后续我们进行更加复杂，更加前沿的项目阶段。

本阶段有四个方向：

!!! tip "概览"

    - 基于勘智 K230 卫星星务计算单元的 QEMU 建模
    - 基于 CXLMemSim 和 QEMU 的 GPU 可观测系统构建
    - 基于大模型 agent 的 QEMU 自动化外设建模
    - 基于 Wine-CE 的 RISC-V 硬件适配 x86 应用/游戏
